# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## TL;DR

- Be kind. Critique ideas, not people.
- Assume good faith. Most "wrong" answers are half-right and worth building on.
- Stay on topic. This repo is about GPU passthrough — not general hardware drama.
- Vendor discussions welcome. Vendor flame-wars not.
- Respect the scope. "My RTX 4090 doesn't work" is a valid question; "NVIDIA sucks" is not.

## Enforcement

Violations may result in warnings, PR/issue closure, or ban from the project. The maintainer has final say on what falls within or outside the project's scope.
